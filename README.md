# GribShooter 2025

A tower defense strategy game demo built with Unity, featuring deck-building mechanics and wave-based combat.

## ğŸ® Game Description

GribShooter 2025 is a strategic tower defense game where players must protect their base from waves of enemies. The game combines traditional tower defense mechanics with deck-building elements, creating a unique gameplay loop where players collect and upgrade cards to improve their defensive capabilities.

### Core Gameplay Loop
- **Defend the Base**: Protect your base within a designated area from enemy waves
- **Combat Options**: 
  - Shoot enemies directly
  - Deploy units to fight
  - Build defensive structures
  - Destroy enemy bases
- **Deck Building**: Collect weapon, unit, and building cards at the end of each wave
- **Progression**: Use collected cards to improve your arsenal for the next wave

## ğŸ¯ Game Features

- **Wave-based Combat**: Face increasingly challenging enemy waves
- **Deck Building System**: Collect and upgrade cards between waves
- **Multiple Defense Strategies**: Direct combat, unit deployment, and building construction
- **Progressive Difficulty**: Each wave becomes more challenging
- **Card Collection**: Unlock new weapons, units, and buildings
